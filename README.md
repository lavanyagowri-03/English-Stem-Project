# 📘 English Grammar Master – Smart Language Learning System

## 📌 Overview  
English Grammar Master is an interactive learning system designed to improve English language skills using a technology-driven approach. This project focuses on key grammar and communication concepts such as Speech, Voice, Prepositions, Tenses, Degrees of Comparison, Modal Verbs, and Truth vs Fake News Identification.

It combines traditional English learning with modern computational methods to create a smarter and more engaging learning experience.

---

## 🎯 Problem Statement  
Students often struggle with understanding core English grammar concepts and identifying reliable information in digital content. Traditional learning methods lack interactivity and real-time feedback.

---

## 💡 Proposed Solution  
This project provides a smart learning platform that:
- Teaches important grammar concepts interactively  
- Helps users practice and improve English skills  
- Identifies whether given information is true or fake  
- Enhances understanding through structured modules and examples

3. Install dependencies

4. Run the project  
python main.py
